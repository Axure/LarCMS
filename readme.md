## LarCMS

A simple CMS based on Laravel.

### Usage

Like any Laravel application:

```bash
git clone https://github.com/Axure/LarCMS.git
cd LarCMS
npm install
gulp
composer install
```

Then (a quick start):

```bash
php artisan serve
```

Or you can configure it with whatever server you like.
